Usage Guide

How to activate and use each feature of the Mayaavi extension

Toggle Proctoring

1

Find and click the DotPrep logo on the test page

2

Check the "End Test" button color:

  • If it has a overlay then proctoring is turned off

When proctoring is OFF, the "End Test" button and editor controls are disabled to prevent accidental submission.

Ghost Writer

1

Copy the text you want to type to your clipboard

2

Method 2: Click the clock icon near the timer

3

Type any key to automatically insert characters from clipboard one by one

Ghost Writer always enables proctoring. The "End Test" button will turn white when active.

Tab Switching

1

Ensure that proctoring is turned off by pressing DotPrep logo.

2

Switch between tabs freely using Alt+Tab or browser tabs

Tab switching works only when proctoring is off, with no flags or warnings.

CodeThief

1

Make sure you're on a coding question page

2

Type .get answer anywhere in the code editor

3

The answer will be automatically copied to your clipboard

4

Use Ghost Writer to paste the answer naturally if proctoring is active

While using CodeThief, some answers may be in a different language. You can use ChatGPT to convert them to the required language.

Auto MCQ Solver

1

Navigate to an MCQ test page with radio buttons

2

Type .solve mcq to activate the MCQ solver

3

Watch as all answers are automatically filled in with correct options

When answers are successfully applied, the "End Test" button briefly flashes green to confirm completion.

Emergency Uninstall

1

If you need to quickly remove the extension in an emergency situation

2

Press Ctrl+Shift+X from any page

3

The extension will be immediately uninstalled from your browser

This is an emergency feature. To reinstall, you'll need to follow the installation process again.

Frequently Asked Questions

Find answers to common questions about Mayaavi

Is using Mayaavi extension allowed by my institution?

The use of Mayaavi might be against your institution's academic integrity policy. This extension is provided for educational purposes, and you should use it at your own discretion and responsibility. Always check your institution's policies regarding test-taking aids.

How do I know if the extension is working properly?

When the extension is active, clicking the DotPrep logo adds an overlay to certain buttons. A overlay indicates that proctoring is disabled. If there's no overlay, proctoring is enabled and the extension is functioning as expected. You can also switch between tabs freely without triggering any flags from the system. Would you like a more formal or more casual tone?

What is Ghost Writer?

Ghost Writer (activated by clicking the clock logo near the timer) enables a special feature that lets you "type" clipboard contents character by character, mimicking manual typing to avoid detection by systems that block traditional paste commands.

Will the extension be detected by the Amypo system?

Mayaavi is designed to work discreetly and bypass detection mechanisms. However, we cannot guarantee that future updates to the Amypo system won't introduce new detection methods.

How do I update to the latest version?

When a new version is available, you'll receive a mail with the new extension To update, remove the current extension from your browser, and install the new version following the same installation steps.

What is the emergency uninstall shortcut?

The emergency uninstall shortcut (Ctrl+Shift+X) allows you to instantly remove the extension from your browser in urgent situations, such as when a proctor doubts you. This feature ensures you can quickly hide any evidence of using the extension. To reinstall after using this feature, you'll need to follow the regular installation steps again.
